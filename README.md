# VocaDex 📖🔴

VocaDex de vocabulaire anglais — PWA pour capturer les mots inconnus et les réviser en flashcards.

**App en ligne : https://destange.github.io/vocadex/**

## Fonctionnalités
- **Capture** : ajoute un mot anglais, l'app récupère définition/phonétique/exemples (dictionaryapi.dev, ou Urban Dictionary pour l'argot)
- **VocaDex** : fiches complètes numérotées, raretés, évolutions selon la maîtrise, liens Reverso Context / WordReference / Urban Dictionary / Cambridge
- **Révision** : flashcards EN→FR et FR→EN, algorithme adaptatif (répétition espacée pondérée par les erreurs)
- **Badges** de progression
- **Sauvegarde** automatique dans un Gist GitHub (token à coller dans Réglages, scope `gist` uniquement)
- **Rappels** de révision (Chrome Android, app installée)

## Installation sur Android
1. Ouvrir https://destange.github.io/vocadex/ dans Chrome
2. Menu ⋮ → « Ajouter à l'écran d'accueil »
3. Dans Réglages : coller le token Gist et activer les rappels
